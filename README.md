# 🔐 Secure Login API

Production-ready Flask backend with user registration, password hashing, and SQLite database. **Demonstrates authentication skills for IT/Backend roles.**

## ✨ Features
- **POST /api/register** - Create users with hashed passwords (Werkzeug)
- **POST /api/login** - Authenticate users with password verification
- **GET /api/users** - List all registered usernames
- SQLite database with UNIQUE username constraint
- JSON REST API responses with proper HTTP status codes

## 🛠 Tech Stack
- **Python 3.8+**, Flask 3.0, Werkzeug 3.0, SQLite3
